## 🧠 6. Binary Search

### 📖 Description
Efficient searching by dividing the search space.

### 🔹 Pattern
```cpp
low = min;
high = max;
while (low < high) {
    int mid = (low + high) / 2;
    if (condition(mid))
        high = mid;
    else
        low = mid + 1;
}
```
